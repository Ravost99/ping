## Features
- Using `#` at the start of line to 'comment' the line out in `sites.txt`
- Ping rounds
- Custom site list
- Ping headers
- Down Log
- Self ping
- Configurations file
- File updates & version updates (like `git pull`)

## Setup & Running
Just run `python3 main.py` 
or `python main.py`<br>
